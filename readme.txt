PixFields offers you a nice interface to add custom metadata into your favorite post type.

== Description ==

With [PixFields](https://github.com/pixelgrade/pixfields) you can have a custom fields visual manager.

*Note:* this plugin does not add any style to the output of custom fields. Its just adding metadata and offers a way to overwrite the html from the theme.
The theme should style the output of the metadata.

== Installation ==

1. Install PixFields either via the WordPress.org plugin directory, or by uploading the files to your `/wp-content/plugins/` directory
2. After activating PixFields go to any edit page and see the PixFields metabox in the right sidebar.
